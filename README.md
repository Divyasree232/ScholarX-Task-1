# Smart Document Classifier AI

An AI-powered Smart Document Classifier built using React, Vite, Machine Learning, and NLP technologies. The system automatically analyzes and classifies uploaded documents such as resumes, invoices, research papers, legal files, and medical reports using intelligent AI models.

---

## Features

* AI-powered document classification
* PDF, DOCX, and image upload support
* OCR text extraction
* NLP-based text processing
* Real-time prediction results
* Confidence score visualization
* Smart search and filtering
* Responsive modern dashboard
* Bright futuristic UI with animations
* Drag-and-drop upload system

---

## Tech Stack

### Frontend

* React.js

### Backend

* Python Flask / FastAPI

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Smart-Document-Classifier.git
cd Smart-Document-Classifier
```

---

## Frontend Setup

```bash
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## Backend Setup

```bash
pip install -r requirements.txt
python app.py
```

Backend runs on:

```bash
http://localhost:5000
```

---

## AI/ML Workflow

1. Upload document
2. Extract text using OCR
3. Apply NLP preprocessing
4. Convert text into vectors using TF-IDF
5. Predict document category using ML model
6. Display classification result with confidence score
